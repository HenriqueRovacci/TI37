🌐 O que é a Web?

A web é um sistema de páginas conectadas por links, acessadas via navegadores. É como uma rede gigante de informações interligadas.

📌 Cabeçalhos HTML

⚙️ Como funciona?

  • 🌍 Você digita o site
  • 📡 O navegador envia a requisição
  • 🖥️ O servidor responde
  • 🎨 O navegador monta a página
